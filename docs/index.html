Fully Polished Features

No AI slop or unnecessary overhead. Designed with pure utility, speed, and premium user experience in mind.

Smart Scout Engine

Dynamically queries Instagram's database to expand search tags into related trending hashtags. Ranks candidates by composite engagement scores to retrieve the best viral content.

Smart Caption Replacer

Automatically replaces the original credits/mentions with your own handle, strips out multi-tag spam lines, and keeps video courtesy and poetry notes intact.

Multi-Auth System

Flexible authentication. Supports standard username/password login, 2FA, session file resumption, and session cookie bypass.

Session Manager

Login sessions are securely saved as local JSON configs. The bot auto-resumes settings on launch, and allows you to delete/remove saved accounts directly from the login menu.

Robust Downloads

Downloads high-res videos and images. Generates standard Instagram-compatible thumbnails using local FFMPEG integration with custom 3x retry policies.

Automated Cleanup

Ensures zero storage bloat. Temporarily saved media downloads and cache artifacts are completely unlinked from your storage post-upload.

The Posting Pipeline

How the bot manages scouting, processing, and publishing completely autonomously.

1

Secure Login & Session Resumption

The bot verifies existing local sessions. If none exist, it logs in using your choice of username/password, file uploads, or cookie values.

2

Target Scouting & Engagement Metric Scoring

Provide multiple target hashtags. The bot fetches top performing items and ranks them with a formula: engagement = (views * 0.3) + (likes * 0.7).

3

Interactive Review Table

A formatted table displays candidates' author, view count, like count, video duration, and caption preview. You select what to post using range syntax (e.g. 1-3,5, or all).

4

Caption Re-Engineering & Asset Uploading

The bot downloads selected media, converts/generates thumbnails, polishes captions (injecting your page's username), uploads them via private Instagram API, and safely cleans up temporary assets.
